Steps to run Test Companion Demo
1. Install the extension from https://marketplace.visualstudio.com/items?itemName=browserstackcom.test-companion
2. Sign in with BrowserStack
3. Within VS code, clone this repository into any folder
4. Run npm install command from terminal
5. To be able to demo running tests on BrowserStack Automate, replace the values in `browserstack.yml` with actual values

With this your initial setup of your workspace to demo Test Companion is ready!
